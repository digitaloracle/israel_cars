Israel Vehicle Lookup

Chrome Extension — query Israeli vehicle registration, mileage & ownership history by license plate.

Instantly look up any Israeli vehicle using the official data.gov.il government open-data API. Results appear in a side panel without leaving the page you're on.

  • Registration details — make, model, year, color, fuel type
  • Last reported mileage
  • Full ownership history timeline
  • Pollution group with visual scale
  • Inspection & registration expiry dates with color coding
  • Recent searches saved locally
